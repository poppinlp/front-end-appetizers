Development environment

Text editor

文本编辑器主要用来进行一些文本编辑工作，是开发中的主力工具。使用自己熟悉或者顺手的文本编辑器即可。

  • Sublime text
  • Atom
  • Nodepad++
  • Vim
  • e.t.c

Integrated development environment

集成开发环境，简称IDE，顾名思义即集成了很多功能的一套开发环境。相比于文本编辑器，更重但功能也更全。

  • Webstorm
  • Aptana
  • Brackets

Execution environment

  • Chrome
  • Firefox + firebug
  • Nodejs
